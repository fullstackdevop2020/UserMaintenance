1. Import project.
2. Run mvn clean build
3. Start UserMaintenanceApplication as java application
4. App URL -  http://localhost:8000
5. Access h2-console -  http://localhost:8000/h2-console/
6. Access swagger to test the endpoints - http://localhost:8000/swagger-ui/#/user-maintenance-rest-controller
7. Monitor App - http://localhost:8000/actuator
